## 1. Pendahuluan

Aplikasi ECR merupakan sistem manajemen emergency/security alert yang digunakan untuk mendukung proses pelaporan, penerimaan, penanganan, pemantauan, dan dokumentasi kejadian darurat. Sistem ini menghubungkan pelapor, staff departemen, admin, superadmin, dan pengguna view only dalam satu alur operasional.

Secara umum, aplikasi memiliki fitur login berbasis role, dashboard operasional, form tambah alert, status alert, riwayat alert, detail report, monitoring TV, master data, approval staff, dan cetak/export laporan.

## 2. Halaman Login

Penjelasan:
Halaman Login digunakan oleh seluruh pengguna untuk masuk ke aplikasi sesuai hak akses masing-masing. Setelah login berhasil, sistem mengarahkan pengguna ke dashboard sesuai role.

Langkah penggunaan:
1. Buka aplikasi ECR.
2. Masukkan username dan PIN/password.
3. Tekan tombol Login.
4. Sistem menampilkan dashboard sesuai role pengguna.

Form Tambah Alert digunakan untuk membuat laporan emergency. User mengisi departemen tujuan, deskripsi kejadian, prioritas, lokasi, dan foto bukti.
